## 📋 Project Overview

This project implements a few-shot object detection system for thermal/infrared imagery, specifically designed for optoelectronic systems in military applications. The system can adapt to new target classes using only 1-10 labeled examples and optional text descriptions.

### Key Features
- **Few-shot learning** with prototypical networks
- **Multi-modal understanding** using CLIP (vision-language)
- **Thermal image preprocessing** for low-contrast imagery
- **Modular design** with OPM (training) and MOC (detection) components
- **Size-optimized** for <100MB deployment
